Project Title: Security Scanner by bhaves malik

Description:
A basic vulnerability scanner built using Python. It scans for open ports, checks basic service info, and identifies possible security risks. Useful for understanding how penetration testers perform reconnaissance on target systems.

Features:
- Scans open ports (common range)
- Shows which services are running
- Identifies common weak ports (like FTP, Telnet, etc.)
- CLI-based tool (command line)

How to Run:
1. Install Python 3
2. Open terminal and run:
   python scanner.py [target_ip]

Example:
   python scanner.py 192.168.1.1
